◆ TempCheck

void MainWindow::TempCheck ( )
slot

MainWindow::TempCheck.

The temperature stores in vtemp_ until the given number is reached. A moving average is calculated from these values. If this moving average is smaller than the threshold value, it is assumed that there is no temperature change and the Quit function is executed. If it is greater than the threshold value, set countTempCheck_ to 0 and exit TempCheck mode. Note that this function does not work if the temperature is in the ignore range given setParamters function.
