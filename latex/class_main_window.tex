\mbox{\hyperlink{class_main_window_af30ebac87cad0ccff631b6e11bcc50c8}{Main\+Window\+::\+Temp\+Check}}. 

The temperature stores in vtemp\+\_\+ until the given number is reached. A moving average is calculated from these values. If this moving average is smaller than the threshold value, it is assumed that there is no temperature change and the Quit function is executed. If it is greater than the threshold value, set count\+Temp\+Check\+\_\+ to 0 and exit Temp\+Check mode. Note that this function does not work if the temperature is in the ignore range given set\+Paramters function.
