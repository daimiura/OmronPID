\doxysubsection{Detailed Description}
Represents two doubles as a mathematical 2D vector. 

This class acts as a replacement for QVector2D with the advantage of double precision instead of single, and some convenience methods tailored for the \mbox{\hyperlink{class_q_custom_plot}{QCustom\+Plot}} library.
