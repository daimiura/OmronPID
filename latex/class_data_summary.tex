{\footnotesize\ttfamily void Data\+Summary\+::write\+Data (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})}



Writes data to file. 

Writes temperature, MV and SV values to the data file. \mbox{\hyperlink{class_the}{The}} data is saved in the following format\+: $<$date and time$>$\textbackslash{}t$<$seconds since epoch$>$\textbackslash{}t$<$temperature$>$\textbackslash{}t$<$\+MV$>$\textbackslash{}t$<$\+SV$>$

\mbox{\hyperlink{class_the}{The}} method checks whether the saving of data has been enabled. If not, the method exits. \mbox{\hyperlink{class_the}{The}} data is written to a file specified by file\+Name\+\_\+. If the file does not exist, a new file is created by calling \mbox{\hyperlink{class_data_summary_a721d29528f5aeb7519a92c0d40d44c45}{generate\+Save\+File()}}. \mbox{\hyperlink{class_the}{The}} temperature, MV and SV values are retrieved by calling \mbox{\hyperlink{class_data_summary_a1b16196dc54f8c0709351613bb884ea8}{get\+Temperature()}}, \mbox{\hyperlink{class_data_summary_a9dbdea2095a1a84c8af0204b554ddadd}{get\+MV()}} and \mbox{\hyperlink{class_data_summary_ac83d8023240b79ab2c9b19e34df69884}{get\+SV()}} respectively. \mbox{\hyperlink{class_the}{The}} date and time of writing the data is obtained using QDate\+Time\+::current\+Date\+Time() method. \mbox{\hyperlink{class_the}{The}} data is written in the text format. Once the data is written, the signal \mbox{\hyperlink{class_data_summary_aebab52d567efb28866ddd986e58f88fa}{File\+Save(true)}} is emitted. \begin{DoxyReturn}{Returns}
