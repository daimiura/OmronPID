This method creates a HTTP POST request to the LINE messaging API using the provided message and token. \mbox{\hyperlink{class_the}{The}} request is sent using the QNetwork\+Access\+Manager class. \mbox{\hyperlink{class_the}{The}} method waits for the reply from the server in a non-\/blocking manner by connecting to the QNetwork\+Access\+Manager\+::finished signal. Once the reply is received, the Notify\+::handle\+Network\+Reply() method is called to process the reply.
